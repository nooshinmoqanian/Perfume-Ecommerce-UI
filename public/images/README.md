این پوشه برای قرار دادن تصاویر استاتیک محصولات است.
نام فایل‌ها را مطابق الگو قرار دهید تا در سایت بارگذاری شوند:

- hero-<productId>.jpg  -> تصویر هدر (قسمت بالای صفحه)
- row-<productId>.jpg   -> تصاویر بخش ویرایش (HomePage)
- discount-<productId>.jpg -> تصاویر کارت‌های تخفیف

مثال: `row-abc123.jpg` یا `discount-42.jpg`

اگر تصویر موجود نباشد، مرورگر با `onError` تصویر جایگزین را بارگزاری می‌کند.
